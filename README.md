# Hi, I'm Jessy 👋

I build **AI-powered automation systems for healthcare organizations** — turning complex administrative workflows into intelligent, hands-off pipelines.

## What I Do

Healthcare admin is drowning in repetitive, high-stakes tasks: patient letters, portal message triage, provider escalations, phone call transcripts, and compliance documentation. I build systems that handle all of it accurately, consistently, and at scale.

My work sits at the intersection of **clinical operations** and **modern AI infrastructure**. I don't just wire APIs together. I design end-to-end architectures that are HIPAA-aware, white-label ready, and built to run without babysitting.

## What I Stand For

- **Healthcare should work smarter** — no more manual letter-writing, no more missed escalations, no more staff burnout on administrative volume
- **AI in healthcare must be responsible** — hard clinical stops, no medical advice, strict PHI minimization, every interaction logged
- **Automation should be explainable** — every workflow has an SOP, every decision is traceable, every failure is a learning loop
